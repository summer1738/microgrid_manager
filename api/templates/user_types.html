User types

Users are the people or systems that interact with, monitor, or control the microgrid.

User type Role
Microgrid Operator Manages and monitors system operation
Energy Consumers Use electricity from the microgrid (households, schools, clinics, small businesses)
Maintenance Engineers Maintain and troubleshoot system
Utility / Grid Operator Manages grid interaction (when microgrid is grid-connected)
1. System Administrator (Microgrid Operator)

Main controller of the system.

  • Monitor system performance
  • View PV generation forecasts
  • Control battery charging/discharging
  • Manage system settings & view alerts

In this system: Uses forecasting module, IEBA (energy management), and monitoring dashboard (UCLPI).

2. Energy Consumers (Households / Buildings)

People and facilities using electricity from the microgrid.

  • Consume electricity
  • View energy usage & availability
  • May see: current usage, solar power, battery level

Examples: Homes, schools, clinics, small businesses.

3. System Maintenance / Engineers

Maintain and troubleshoot the system.

  • Maintain solar panels & check battery health
  • Monitor system faults & update software
  • Check logs, diagnose failures, run diagnostics
4. Utility / Grid Operator (optional)

Relevant when the microgrid connects to the main grid.

  • Monitor power exchange
  • Manage grid connection
  • Regulate energy flow
