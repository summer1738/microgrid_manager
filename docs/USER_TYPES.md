# User Types — Microgrid Management System

Users are the people or systems that interact with, monitor, or control the microgrid. They are grouped into the following types.

---

## 1. System Administrator (Microgrid Operator)

The main controller of the system.

| Aspect | Description |
|--------|-------------|
| **Responsibilities** | Monitor system performance, view PV generation forecasts, control battery charging/discharging, manage system settings, view system alerts |
| **Example actions** | Check solar generation forecast, monitor battery state of charge, adjust microgrid control parameters, export system data |
| **In this system** | Interacts with: **forecasting** module, **energy management controller** (IEBA), **monitoring dashboard** (UCLPI) |

---

## 2. Energy Consumers (Households / Buildings)

People and facilities using electricity from the microgrid.

| Aspect | Description |
|--------|-------------|
| **Examples** | Homes, schools, clinics, small businesses |
| **What they do** | Consume electricity, view their energy usage, possibly see energy availability |
| **Example information** | Current energy usage, available solar power, battery level, energy cost |

---

## 3. System Maintenance / Engineers

Users who maintain and troubleshoot the system.

| Aspect | Description |
|--------|-------------|
| **Responsibilities** | Maintain solar panels, check battery health, monitor system faults, update software |
| **Example actions** | Check system logs, diagnose equipment failures, run system diagnostics |

---

## 4. Utility / Grid Operator (Optional)

Relevant when the microgrid connects to the main grid.

| Aspect | Description |
|--------|-------------|
| **Responsibilities** | Monitor power exchange, manage grid connection, regulate energy flow |

---

## Summary Table (for report)

| User Type | Role |
|-----------|------|
| **Microgrid Operator** | Manages and monitors system operation |
| **Energy Consumers** | Use electricity from the microgrid |
| **Maintenance Engineers** | Maintain and troubleshoot system |
| **Utility/Grid Operator** | Manages grid interaction (if grid-connected) |
